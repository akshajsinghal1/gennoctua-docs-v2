## What you're adding

You're adding a "Try On" button to a fashion brand's product page. When a shopper clicks it, they upload a photo and the AI shows them wearing the product. It takes about 20 seconds to generate.

---

## What you need before starting

<Steps>
  <Step title="A brand key">
    Get this from Gennoctua. It looks like `gn_pk_yourbrand_xxx`
  </Step>
  <Step title="A website">
    Any website that shows products (Shopify, Webflow, plain HTML, React, etc.)
  </Step>
  <Step title="A product image URL">
    The URL of the product image on the page
  </Step>
</Steps>

That's it. No backend, no server setup, no API keys to manage.

---

## The simplest way to add it

Copy and paste this code anywhere on your product page. Change the three values marked with arrows:

```html
<script src="https://cdn.jsdelivr.net/gh/akshajsinghal1/gennoctua-sdk-v2@main/dist/cdn/personalize.min.global.js"></script>

<div>
  <p>Upload your photos to try this on:</p>
  <input id="photos" type="file" multiple accept="image/*" webkitdirectory />
  <br /><br />
  <button id="tryOnBtn" disabled>Try On</button>
  <p id="status"></p>
  <img id="result" style="display:none; max-width:500px; border-radius:8px" />
</div>

<script>
  // CHANGE THIS: your brand key from Gennoctua
  const BRAND_KEY = "gn_pk_yourbrand_xxx";

  // CHANGE THIS: the URL of the product image
  const PRODUCT_IMAGE_URL = "https://yourbrand.com/product.jpg";

  // CHANGE THIS: what type of product it is (see list below)
  const PRODUCT_TYPE = "sunglasses";

  // --- Don't change anything below this line ---

  let sdk;
  window.Personalize.init({ auth: { publicKey: BRAND_KEY } })
    .then(s => { sdk = s; });

  document.getElementById("photos").addEventListener("change", async (e) => {
    document.getElementById("status").textContent = "Reading your photos...";
    const summary = await sdk.ingestImages(e.target.files, p => {
      document.getElementById("status").textContent = p.message;
    });
    if (summary.availableCategories.length > 0) {
      document.getElementById("status").textContent = "Ready! Click Try On.";
      document.getElementById("tryOnBtn").disabled = false;
    } else {
      document.getElementById("status").textContent = "Couldn't find a face in your photos. Try different ones.";
    }
  });

  document.getElementById("tryOnBtn").addEventListener("click", async () => {
    document.getElementById("status").textContent = "Generating your look (~20 seconds)...";
    document.getElementById("tryOnBtn").disabled = true;
    try {
      const result = await sdk.personalize({
        imageUrl: PRODUCT_IMAGE_URL,
        productType: PRODUCT_TYPE,
        gender: "female"
      });
      document.getElementById("result").src = result.imageUrl;
      document.getElementById("result").style.display = "block";
      document.getElementById("status").textContent = "Here's how it looks on you!";
    } catch (err) {
      document.getElementById("status").textContent = "Something went wrong. Try again.";
      document.getElementById("tryOnBtn").disabled = false;
    }
  });
</script>
```

## What to change

### 1. Brand Key
```js
const BRAND_KEY = "gn_pk_yourbrand_xxx";
//                 replace with your actual brand key from Gennoctua
```

### 2. Product Image URL
```js
const PRODUCT_IMAGE_URL = "https://yourbrand.com/product.jpg";
//                         right-click the product image -> "Copy image address"
```

### 3. Product Type
```js
const PRODUCT_TYPE = "sunglasses";
//                    pick from the list below
```

**Product type options:**

| What you're selling | Use this |
|---|---|
| Sunglasses | `"sunglasses"` |
| Eyeglasses / reading glasses | `"eyeglasses"` |
| Men's clothing | `"mens_clothing"` |
| Women's clothing | `"womens_clothing"` |
| Kids' clothing | `"kids_clothing"` |
| Shoes / sneakers | `"footwear"` |
| Necklaces | `"jewellery"` |
| Earrings | `"earrings"` |
| Handbags / purses | `"bags"` |
| Lipstick | `"makeup_lipstick"` |
| Foundation | `"makeup_foundation"` |
| Mascara | `"makeup_mascara"` |

## How to add it using AI

Paste this into your AI tool (Cursor, ChatGPT, Lovable, etc.):

```
I want to add a virtual try-on feature to my product page.

Here is the code I need to add:
[paste the complete code block from above]

My brand key is: gn_pk_yourbrand_xxx
My product image URL is: [your product image URL]
My product type is: [sunglasses / clothing / footwear / etc]

Please:
1. Add this code to my product page
2. Replace the three values at the top (BRAND_KEY, PRODUCT_IMAGE_URL, PRODUCT_TYPE)
3. Style the upload button and result image to match my existing design
4. Place the try-on section below the product image
```

## Platform-specific instructions

<AccordionGroup>
  <Accordion title="Shopify">
    1. Go to your Shopify admin
    2. Online Store -> Themes -> Edit code
    3. Open `sections/product.liquid`
    4. Find the closing `</div>` near the product image
    5. Paste the code there
    6. Change the three values at the top
    7. Save and preview

    For the product image URL in Shopify, use:
    ```js
    const PRODUCT_IMAGE_URL = "{{ product.featured_image | img_url: 'master' }}";
    ```
  </Accordion>
  <Accordion title="Webflow">
    1. Open your product page in Webflow Designer
    2. Add an "Embed" element where you want the try-on button
    3. Paste the full code into the embed
    4. Change the three values
    5. Publish
  </Accordion>
  <Accordion title="Plain HTML">
    1. Open your HTML file
    2. Find your product image tag: `<img src="..." />`
    3. Paste the code right after the image tag
    4. Change the three values
    5. Save and open in browser
  </Accordion>
</AccordionGroup>

## Testing it

1. Open your page in a browser
2. Click the file input and select a folder with your photos
3. Wait a few seconds while it scans the photos
4. When it says "Ready! Click Try On" - click the button
5. Wait ~20 seconds
6. Your try-on image should appear

**Good photos to upload:**
- Clear, front-facing face
- Good lighting (not too dark, not too bright)
- No sunglasses in the photo
- At least one photo, up to 80

## Troubleshooting

<AccordionGroup>
  <Accordion title='"Couldn&apos;t find a face in your photos"'>
    Try uploading photos where your face is clearly visible, front-facing, good lighting.
  </Accordion>
  <Accordion title='"Something went wrong. Try again"'>
    Try clicking the button again. If it keeps failing, check your brand key is correct.
  </Accordion>
  <Accordion title="Nothing happens when I click the button">
    Make sure you uploaded photos first and waited for the "Ready!" message.
  </Accordion>
  <Accordion title="The image URL is broken">
    Right-click your product image -> Copy Image Address -> paste that URL as `PRODUCT_IMAGE_URL`.
  </Accordion>
  <Accordion title="Still stuck?">
    Share your code with your AI assistant and ask: "The Gennoctua try-on isn't working. Here is my code: [paste code]. The error is: [paste error message]."
  </Accordion>
</AccordionGroup>
