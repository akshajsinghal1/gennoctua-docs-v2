## Core Workflow

```ts
// 1. Initialize SDK
const sdk = await Personalize.init({ auth: { ... } });

// 2. Subscribe to events (optional but recommended)
sdk.on("personalization:completed", ({ imageUrl }) => {
  document.getElementById("result").src = imageUrl;
});

sdk.on("personalization:failed", ({ error }) => {
  console.error("Try-on failed:", error);
});

// 3. Ingest user photos (from file input)
const fileInput = document.getElementById("photoInput");
fileInput.addEventListener("change", async (e) => {
  const summary = await sdk.ingestImages(e.target.files, (progress) => {
    console.log(progress.message); // "Scanning faces 3 of 14..."
  });
  console.log("Available categories:", summary.availableCategories);
});

// 4a. Single product
const result = await sdk.personalize({
  imageUrl: "https://brand.com/product.jpg",
  productType: "sunglasses",
  gender: "female",
  productId: "sku-123"
});

console.log(result.imageUrl);   // personalized image URL
console.log(result.cacheHit);   // true if returned from cache
console.log(result.jobId);      // HP job ID

// 4b. Multiple products (product listing page)
await sdk.personalizeAll([
  { productId: "sku-1", imageUrl: "https://brand.com/glasses1.jpg", productType: "sunglasses", gender: "female" },
  { productId: "sku-2", imageUrl: "https://brand.com/glasses2.jpg", productType: "sunglasses", gender: "female" },
  { productId: "sku-3", imageUrl: "https://brand.com/shirt.jpg",    productType: "mens_clothing", gender: "male" },
], (result) => {
  if (result.status === "completed") {
    document.getElementById(result.productId).src = result.imageUrl;
  }
});
```

## Configuration Options

```ts
Personalize.init({
  // REQUIRED
  auth: ProxyAuthConfig | PublicKeyAuthConfig,

  // Optional â€” controls which AI pipelines run during ingestImages().
  // Tells the SDK what category of products the brand sells.
  //
  //   "eyewear" | "fashion" | "footwear" | "jewellery" | "bags" | "makeup"
  //       â†’ face detection only (room classifier never loads)
  //   "furniture"
  //       â†’ room classifier only (face detection skipped)
  //   "all" (default)
  //       â†’ both pipelines run on all uploaded photos
  //
  // Pass an array for multi-category brands:
  //   personalizationMode: ["fashion", "jewellery", "makeup"]
  personalizationMode: "fashion", // or "furniture" | "all" | ["fashion", "furniture"]

  // Product detection
  product: {
    imageSelector: ".product-image",
    gallerySelector: ".product-gallery",
    productType: "sunglasses",
    rules: [
      { match: { urlPattern: "/sunglasses/" }, productType: "sunglasses" }
    ],
    detectFromStructuredData: true,
    detectFromDom: true,
  },

  // Cache settings
  cache: {
    resultTtlMs: 604800000,       // 7 days (default)
    selectionTtlMs: 86400000,     // 24h (default)
    restoreActiveJobs: true,
    activeJobMaxAgeMs: 300000,    // 5 min (default)
  },

  // Rate limiting
  rateLimit: {
    personalization: {
      enabled: true,
      cooldownMs: 10000,
      maxPerSession: 20,
      maxPerProduct: 3,
      singleFlight: true,
    }
  },

  // Analytics
  analytics: {
    enabled: true,
    onEvent: (event) => {
      mixpanel.track(event.eventName, event);
    }
  },

  debug: false,
  maxImages: 80,
  pollIntervalMs: 1500,
  pollMaxAttempts: 120,
});
```

## Methods

### `sdk.ingestImages(files, onProgress?)`

Processes a FileList from a folder/file input. For **person products** runs face detection, gender classification, and pose scoring. For **furniture products** runs on-device room detection (YOLOv8) to classify photos as bedroom, living room, or dining room. An LLM refinement pass picks the single best photo per category.

```ts
const summary = await sdk.ingestImages(
  fileInputElement.files,
  (progress) => {
    // progress.message â€” human-readable status string
    // progress.step â€” current step number
    // progress.total â€” total steps
    updateProgressBar(progress.message);
  }
);

// summary.availableCategories â€” e.g. ["female_full_body", "female_face_closeup"]
// summary.missingCategories â€” categories not found
// summary.totalUploaded â€” total valid photos processed
// summary.totalSelected â€” number of profiles selected
```

### `sdk.personalize(opts)`

Submits a single try-on job and returns the result. For multiple products use `sdk.personalizeAll()`.

```ts
const result = await sdk.personalize({
  imageUrl: “https://brand.com/product.jpg”,  // REQUIRED
  productType: “sunglasses”,                   // REQUIRED
  gender: “female”,                            // REQUIRED — “male” | “female” | “kid_boy” | “kid_girl”
  productId: “sku-123”,                        // optional
  abortSignal: controller.signal,              // optional
});

// result.imageUrl — personalized image URL
// result.cacheHit — true if returned from cache (instant)
// result.jobId    — job ID for debugging
```

### `sdk.getEligibility(productType?)`

Check if a try-on can be performed before starting.

```ts
const eligibility = await sdk.getEligibility("sunglasses");

if (eligibility.eligible) {
  // show Try On button
} else {
  // eligibility.reason:
  // "REQUIRED_USER_IMAGE_MISSING"
  // "PRODUCT_TYPE_NOT_FOUND"
  // "PRODUCT_TYPE_UNSUPPORTED"
  console.log("Can't try on:", eligibility.reason);
}
```

### `sdk.on / sdk.off`

```ts
// Returns an unsubscribe function
const unsub = sdk.on("personalization:completed", ({ imageUrl, cacheHit, jobId }) => {
  showResult(imageUrl);
});

// Unsubscribe manually
sdk.off("personalization:completed", handler);

// Or use the returned unsubscribe function
unsub();
```

See [API Reference](/sdk/api-reference) for all events.

## Error Handling

```ts
import { SDKError } from "@gennoctua/personalize-core";

try {
  const result = await sdk.personalize({ imageUrl, productType, gender: "female" });
} catch (err) {
  if (err instanceof SDKError) {
    switch (err.code) {
      case "AUTH_INVALID":
        // token expired â€” refresh and retry
        break;
      case "RATE_LIMITED_CLIENT":
        // user is trying too fast
        break;
      case “PERSONALIZATION_JOB_FAILED”:
        // AI generation failed — retry
        break;
      case “PERSONALIZATION_JOB_TIMEOUT”:
        // took too long — retry
        break;
      case "NETWORK_ERROR":
        // check internet connection
        break;
    }
    console.log("Recoverable:", err.recoverable);
  }
}
```

```tsx
function ProductPage() {

  const { ingest, summary, ingesting } = useImageIngestion();
  const { personalize, result, loading, error } = usePersonalization();

  return (
    <div>
      <input type="file" multiple onChange={(e) => ingest(e.target.files)} />
      <button onClick={() => personalize({ imageUrl, productType: "sunglasses" })}>
        Try On
      </button>
      {result && <img src={result.imageUrl} />}
    </div>
  );
}
```

## Caching Behavior

Results are cached in **IndexedDB** (browser storage). Cache key:
```
orgId + userImageHash + productImageHash + productType
```

- Same user + same product = instant result (cache hit)
- Cache survives page refresh and browser close
- Default TTL: 7 days
- Clear cache: `sdk.cache.clearAll()`
