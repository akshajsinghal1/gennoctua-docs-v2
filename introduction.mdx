## How it works

<Steps>
  <Step title="Install the SDK">
    Add via npm for React/Vue/Vite projects, or use the CDN script tag for plain HTML, Shopify, and Webflow.
  </Step>
  <Step title="Initialize with your credentials">
    Use a brand key (frontend-only) or your own backend proxy (full-stack). See [Which integration should I use?](/integration/choose)
  </Step>
  <Step title="Let shoppers upload photos">
    Call `sdk.ingestImages()` â€” the SDK automatically picks the best photo using on-device AI. For person products (fashion, eyewear) it detects faces and scores poses. For furniture products it detects room type from the photo.
  </Step>
  <Step title="Run the try-on">
    Call `sdk.personalize()` with a product image URL and type â€” the AI generates the result in ~20 seconds.
  </Step>
  <Step title="Show the result">
    Display the returned image URL however you like â€” modal, side-by-side, or inline on the product page.
  </Step>
</Steps>

## Two integration modes

### Frontend-Only (no backend needed)

<CodeGroup>
```ts npm
import { Personalize } from "@gennoctua/personalize-core";

const sdk = await Personalize.init({
  auth: {
    publicKey: "gn_pk_yourbrand_xxx"
  }
});
```

```html Script tag
<script src="https://cdn.jsdelivr.net/gh/akshajsinghal1/gennoctua-sdk-v2@main/dist/cdn/personalize.min.global.js"></script>
<script>
  const sdk = await window.Personalize.init({
    auth: {
      publicKey: "gn_pk_yourbrand_xxx"
    }
  });
</script>
```
</CodeGroup>

### Full-Stack (you have a backend)

```ts
const sdk = await Personalize.init({
  auth: {
    proxyUrl: "https://yourbrand.com/api/tryon",
    getToken: async () => await getSessionToken()
  }
});
```
